# U.S. Civics Education Website Documentation

## Website URL

## Overview
The U.S. Civics Education website is a comprehensive online platform that transforms the extensive civics curriculum into an interactive, searchable, and user-friendly educational resource. The website provides access to all 65 lessons across 7 chapters covering essential American government and citizenship concepts.

## Navigation Guide

### **Getting Started**
1. Visit https://yzclbufi.manus.space
2. Click "Start Learning" to begin with Chapter 1
3. Or use "Search Topics" to find specific content

### **Browsing Content**
- Use the sidebar to navigate between chapters and lessons
- Click chapter titles to expand and see all lessons
- Use breadcrumb navigation to understand your location
- Navigate between lessons using previous/next buttons

### **Searching**
- Click the search icon in the header or visit the search page
- Enter keywords to find relevant content
- Use the chapter filter to narrow results
- Click on search results to go directly to lessons

### **Customization**
- Toggle between light and dark modes using the moon/sun icon
- Sidebar automatically expands on larger screens
- Mobile users can access the sidebar via the menu button
